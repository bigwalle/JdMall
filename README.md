# 用weex模仿京东商场页面
# 1、安装node环境
# 2、安装weex $npm install -g weex-toolkit
# 3、启动项目，切换至项目的目录,npm install ；安装:npm i weex-ui -S 执行npm start
# 4、访问ip:端口
# 5、访问ip：端口/web/preview.html 手机端下载weexplaygroud app 并扫码；
